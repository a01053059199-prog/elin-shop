# ELIN 쇼핑몰 실행 안내

## 실행

`start-elin.bat`을 더블클릭하거나 아래 명령을 실행하세요.

```powershell
npm start
```

서버가 켜지면 브라우저에서 아래 주소로 접속합니다.

- 쇼핑몰: http://localhost:4173/
- 주문서: http://localhost:4173/checkout.html
- 관리자: http://localhost:4173/admin.html

## 도메인 적용

실제 도메인 연결 순서는 [DEPLOY.md](./DEPLOY.md)를 확인하세요.

## 관리자

기본 관리자 계정은 아래와 같습니다.

- 아이디: `admin`
- 비밀번호: `1234`

실제 배포할 때는 서버 환경변수 `ADMIN_USER`, `ADMIN_PASSWORD`로 바꾸세요.

## 현재 가능한 기능

- 상품 목록 조회
- 상품 검색/필터
- 장바구니 저장
- 주문서 작성
- 주문 JSON 저장
- 관리자 상품 등록/수정/삭제
- 관리자 주문 확인
- 주문 상태 변경

## 아직 필요한 실제 판매 기능

- 카드 결제 PG 연동
- 회원가입/로그인
- 개인정보처리방침/이용약관 페이지
- 도메인, SSL, 서버 배포
- 사업자등록/통신판매업 신고 정보 반영
